# **App Name**: FREMN Landing

## Core Features:

- Hero Section: Display headline and supporting text to introduce FREMN and its mission.
- Products Section: Showcase current and upcoming AI products with descriptions and status labels.
- Vision Section: Present FREMN's vision in a calm, paragraph-style format.
- Team Section: List team members and their roles, highlighting their collaborative history.
- Footer Section: Display minimal footer content including contact information and links.
- Scroll-Driven Content Reveal: Implement smooth content reveal as the user scrolls through each section of the single-page website.
- Dynamic AI-Powered Tone Adjuster: Utilize an AI tool to analyze and adjust the tone of website copy to ensure it remains calm, confident, and technical, aligning with FREMN's brand identity. The AI tool uses an existing set of values and modifies words that convey wrong sentiment.

## Style Guidelines:

- Background color: Deep charcoal (#121212) for a premium, near-black feel.
- Primary accent color: Muted steel blue (#607B96) for subtle highlights and links.
- Secondary accent color: Soft cyan (#A0D0DE) for secondary interactive elements.
- Text color: Off-white (#F5F5F5) for primary text, and light gray (#A8A8A8) for supporting text.
- Primary font: 'Inter' (sans-serif) for headings and body text to maintain a clean and modern aesthetic.
- Secondary font: 'JetBrains Mono' (monospace) for status labels and metadata accents.
- Each section utilizes a full viewport height (min-h-screen) and vertically centered content with generous whitespace.
- Soft, slow fade and upward movement animations triggered once per section reveal on scroll for a premium feel.
- Implement smooth scrolling behavior without parallax or scroll hijacking.
